Vitamix

This is a blog supposed for marketing. This blog have multi pages, namely: Home Page, About Page, and Details Page. Created using HTML CSS Javascript and Jquery, with mobile-first approach. Flex-box is very reliable for styling this blog, flex-box make this blog perfectly responsive, not only good just for mobile-tablet-pc view, but wider flexibility.

I put in a couple of css tricks in this project, for example: you can see the jumbotron is kinda being a full wide backgroud at the same time, if you're on desktop you can try to resize the window using your pointer, you will see it's always look prefectly-fit with the media screen size. And also if you're on mobile, you will notice on the right side of the screen, you will see there's a little button that will expanded out the hidden category list when you tap on it.

Here's the link to try this website : https://herirudini.github.io/vitamix/

You can also check the source code : https://github.com/herirudini/vitamix
