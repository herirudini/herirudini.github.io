    Computer

    Sejak usia pra-sekolah saya dibelikan komputer oleh orang tua saya, hal itu menjadikan saya memiliki minat dan kegemaran yang sangat tinggi dalam bidang komputer. Sejak SMP saya sudah belajar meng-install ulang dan memperbaiki aplikasi-aplikasi komputer saya secara mandiri. Bahkan saya sudah mampu melakukan software ulang HP Nokia jadul saya bermodalkan USB dan petunjuk di internet. Begitu tinggi rasa penasaran saya terhadap hal-hal seputar komputer, bahkan saya sempat membeli buku pelajaran tentang Visual Basic 6.0 karena penasaran bagaimana cara membuat anti-virus.

    Namun hobby tersebut sempat putus pada saat menjelang masuk SMA. Hal ini dikarenakan pada masa-masa tersebut sedang ramai geng motor, sehingga orang tua saya melarang saya untuk masuk SMK Komputer dan memilih untuk mengirim saya ke pesantren.

    Pada masa kuliah, saya kembali menggeluti hobi saya tersebut, saya kembali mengoprek komputer. Apabila dijabarkan, berikut skill-skill yang telah saya pelajari dalam hobby ini:
    - Software maintenance & troubleshooting (Windows/Ubuntu/Android)
    - Hardware maintenance & troubleshooting (PC/Laptop/Android)
    - Office (Libre/Microsoft)
    - Set-up jaringan (Lokal)
    - HTML, CSS, Javascript

    Music & BMX

    Selain komputer, saya juga memiliki hobby di bidang musik, saat kuliah saya sempat nge-band sebagai bassist, saya mempelajari hal hal yg berkenaan dengan musik seperti, scale, riff, effect, rythm, dll. Saya juga sempat ngamen di beberapa tempat di sekitar kampus. Selain itu saya juga memiliki hobby di bidang olahraga, saya senang sekali bermain BMX dan Skateboarding. Akan tetapi saya tidak begitu antusias dalam dua hobby ini, saya lebih betah mengotak-atik komputer.
